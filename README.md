# WinHook

## Aim:
An application to monitor/log Windows System Calls in order to understand malicious behaviour.

## Build Pre-Requisites:
- Visual Studio Code
- Windows 32bit system
- [Microsoft Detours Library] (https://github.com/Microsoft/Detours)
